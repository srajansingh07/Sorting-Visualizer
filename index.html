Algorithm Learning Hub

Explore sorting algorithms with interactive tutorials and detailed explanations.

Bubble Sort

Easy
Time O(n²)
Space O(1)

Simple comparison-based algorithm that repeatedly steps through the list.

Selection Sort

Easy
Time O(n²)
Space O(1)

Finds the minimum element and places it at the beginning of the unsorted portion.

Insertion Sort

Easy
Time O(n²)
Space O(1)

Builds the sorted array one element at a time by inserting elements into their correct position.

Merge Sort

Medium
Time O(n log n)
Space O(n)

Divide-and-conquer algorithm that divides the array and merges sorted subarrays.

Quick Sort

Medium
Time O(n log n)
Space O(log n)

Selects a pivot element and partitions the array around it recursively.

Heap Sort

Hard
Time O(n log n)
Space O(1)

Uses a binary heap data structure to sort elements efficiently.

Radix Sort

Hard
Time O(nk)
Space O(n+k)

Non-comparative sorting algorithm that sorts by individual digits or characters.

Counting Sort

Medium
Time O(n+k)
Space O(k)

Counts occurrences of each element to determine sorted positions.
